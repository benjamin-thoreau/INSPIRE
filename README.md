# INSPIRE
**IaNalumab in Sjögren's Pulmonary Involvement: Response Evaluation**

---

## About the Project

The **INSPIRE** project is a documentation repository dedicated to storing, organizing, and managing PDF source documents related to the clinical evaluation of IaNalumab in patients with Sjögren's Syndrome-associated pulmonary involvement.

This repository serves as a centralized knowledge base for:
- Clinical study protocols and amendments
- - Scientific literature and bibliographic references
  - - Regulatory documents and submission materials
    - - Study reports and data outputs
     
      - ---

      ## Repository Structure

      ```
      INSPIRE/
      ├── docs/
      │   ├── sources/         # Primary PDF source documents
      │   └── references/      # Bibliographic references & supporting literature
      └── README.md
      ```

      ---

      ## Folder Descriptions

      ### `docs/sources/`
      Contains all primary PDF source documents related to the INSPIRE study, including:
      - Clinical study protocols
      - - Informed consent forms
        - - Case report forms (CRFs)
          - - Clinical study reports (CSRs)
            - - Statistical analysis plans
             
              - ### `docs/references/`
              - Contains bibliographic references and supporting scientific literature, including:
              - - Published peer-reviewed articles
                - - Guidelines and recommendations
                  - - Background scientific documents
                   
                    - ---

                    ## How to Upload PDF Documents

                    1. Navigate to the appropriate folder (`docs/sources/` or `docs/references/`)
                    2. 2. Click **Add file > Upload files**
                       3. 3. Drag and drop your PDF file(s) or click to browse
                          4. 4. Add a descriptive commit message
                             5. 5. Click **Commit changes**
